## 🎯 What You Have

A complete interactive web app to ask someone out on a date with:
- ✨ Beautiful animated letter

- 🏃 Escaping NO button
- 📅 Date & place selection
- 📧 Email notifications (optional)
- 📱 Fully responsive design

---

## 🚀 Get It Online in 3 Steps

### Step 1: Create GitHub Repo
```
Visit https://github.com/new
- Name: ask-her-out
- Select: Public
- Create!
```

### Step 2: Upload Files
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/ask-her-out.git
git branch -M main
git push -u origin main
```

### Step 3: Enable GitHub Pages
```
Settings → Pages → Deploy from branch (main) → Save
```

### ✅ Done! Your app is live at:
```
https://YOUR_USERNAME.github.io/ask-her-out/
```

## 💬 Customization Tips

### Change the Letter Message
Edit in `index.html` around line 29:
```html
<p class="message">
    Your custom message here...
</p>
```

### Change Colors
Edit in `styles.css`:
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### Change Button Text
Edit in `index.html`:
```html
<button class="btn btn-yes" id="yesBtn">YES! 💕</button>
```

## 📧 Email Setup (Optional but Cool!)

1. Go to **emailjs.com** → Sign up (free)
2. Create email service & template
3. Get Service ID, Template ID, Public Key
4. When app opens, enter these credentials
5. Done! You'll get email notifications

## 📱 Test Before Sharing

1. Open `index.html` in browser locally
2. Test on mobile (use Chrome DevTools)
3. Try hovering NO button (it should run away!)
4. Submit a test date
5. Perfect? Share the live link!

## ❓ Quick FAQs

**Q: Does it cost anything?**
A: No! GitHub Pages is free.

**Q: Will she see my code?**
A: Only if you make the repo public. Change privacy in Settings if needed.

**Q: Can I change it after deployment?**
A: Yes! Edit files, commit, and push. Updates in 1-2 minutes.

**Q: My NO button should escape - is it working?**
A: Yes! Hover over the NO button and it jumps away randomly.
